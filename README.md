# Coast-FIRE 经济独立，提前退休

> FIRE运动（Financial Independence, Retire Early；即经济独立，提前退休）是一种以经济独立和提前退休为目标，以及重视幸福感多于对物质的满足的一种生活方式。


> 什么是Coast-FIRE？
> 简单来说，Coast FIRE是指你的退休账户中有足够的资金，无需额外缴款，你的净资产就能增长到按传统退休年龄支持退休。
> 
> 当您的净资产超过我们称之为Coast FIRE数字的水平后，您仍然需要每月赚取足够的开销，但您不再需要为传统退休而储蓄。利用历史验证的增长和通胀率，你可以让现有投资的复利增长来承担重任。这意味着你可以自由选择薪水更低的工作、转为> 兼职工作，或者拥有更多零花钱享受生活。
>
